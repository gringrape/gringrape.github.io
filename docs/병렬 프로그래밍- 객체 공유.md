## 주제
- 객체를 스레드 안전하게 사용하는 방법은 무엇인가?

## 주요 내용
- 불변한 객체를 만든다. -> 상태가 변화하지 않음 -> 정확히는 읽기 전용 객체.
- 객체를 특정 스레드에 한정한다. 
- 객체를 공개하지 않고 숨긴다. 
- 객체를 공개하는 경우에는 안전하게 공개한다. 
- 동기화 처리를 한다.
