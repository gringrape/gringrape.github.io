## 트리에 관해
- binary search tree 같은 경우는 노드들이 규칙을 가지고 있음. 
- 탐색시간이 `log n`. 100억이라는 숫자도 log 값은 10인 것 처럼, 데이터 크기가 커질 수록 linear time과 log time의 편차가 매우 커짐을 알아야 함. 
- 정렬된 키를 가진 원소들을 담고 싶을 때 사용하게 됨.
