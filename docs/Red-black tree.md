- Re-balancing
  - 각 노드는 색깔에 해당하는 자료를 가지고 있고, 이것이 balanced를 확보하는데 쓰임. 
  - Re-balancing으로 인해서 탐색시간이 상수 인 것이 보장됨. 
- 활용
  - Java HashMap에서 bucket의 원소가 일정 개수 이상일때의 자료구조 형식. 이 자료구조를 쓰는 이유는 반복적인 삽입에도 탐색시간을 log로 유지하기 위함.
